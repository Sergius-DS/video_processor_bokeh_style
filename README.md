# video_processor_bokeh_style

Lo que comenzó como una pregunta curiosa mientras veía un tutorial, se convirtió en un motor de procesamiento de video completo. Este repositorio documenta mi camino para entender y replicar los filtros de videollamadas, priorizando la calidad visual y el análisis de rendimiento. Es la prueba de que un problema cotidiano puede ser la mejor oportunidad para aplicar ingeniería de datos y visión artificial de alto nivel.
